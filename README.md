# 犬卫士：基于犬种识别的寄生虫防治智能规划系统

  本项目基于昇思 MindSpore 框架，采用 YOLO目标检测模型，对 Kaggle 斯坦福犬数据集内 20580 张图像开展训练，实现120类细粒度犬种的精准识别与分类。
  
  项目在 Coze 或者n8n等平台搭建专属Agent，借助通讯协议将边缘推理结果融入Agent工作流：将模型输出的犬种标签与用户基本信息同步传入经 Lora 微调的 qwen-2.5-7b-instruct模型，调用“先判犬种、再判虫患风险”核心逻辑，生成包含犬种专业知识与个性化驱虫/防病指南。最终以自然语言文本形式反馈给用户，形成 “图像输入→犬种识别→虫患风险推理→知识/防病方案输出”的无缝闭环，为宠物科普、智能问答、云养宠及宠物医疗健康等场景提供可直接落地的 AI 解决方案。
